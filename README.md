# 🛠️ Projects

A collection of development projects spanning web, mobile, and automation — built with TypeScript, Dart, JavaScript, and more.

---

## 📁 Projects

### 🔗 Jira Integration
Connects with the Jira API to streamline project management workflows. Automates task tracking and issue syncing directly from code.

### 📄 CSV → Markdown
A utility tool that converts CSV data into clean, formatted Markdown tables. Useful for generating docs, reports, or changelogs from raw data.

### 📱 Flutter Form App
A cross-platform mobile form application built with Flutter/Dart. Handles user input with a clean, responsive UI across iOS and Android.

### 🌐 TechAus Landing Page
A landing page for TechAus, built with HTML/CSS/JS. Focused on a polished web presence and fast load performance.

### 🕷️ Telcel Scraper
A web scraper targeting Telcel (Mexican telecom) to extract and structure data — useful for price tracking, plan comparisons, or data pipelines.
